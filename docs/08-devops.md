# 08) البنية التشغيلية (DevOps)

## البيئات
- تطوير (Dev)
- اختبار (Test)
- إنتاج (Prod)

## CI/CD
- بايبلاين لكل خدمة.
- بوابات جودة قبل الدمج والنشر.

## المراقبة
- Logs + Metrics + Traces مركزية.
- تنبيهات للأخطاء الحرجة وتدهور الأداء.

## الاستعادة
- نسخ احتياطي دوري لقواعد البيانات والملفات.
- تجارب استعادة دورية للتأكد من الجاهزية.
