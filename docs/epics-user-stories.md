# Epics وUser Stories

## Epic 1: الهوية والحسابات
- Story: كمستخدم أريد تسجيل دخول موحد للوصول لكل الخدمات.
  - معيار قبول: نجاح تسجيل الدخول وإسناد الدور الصحيح.
- Story: كمدير أريد إدارة صلاحيات الأدوار.
  - معيار قبول: منع الوصول غير المصرح لكل مسار.

## Epic 2: يارا (OCR + تقييم)
- Story: كطالب أريد رفع ورقة والحصول على تقييم أولي.
  - معيار قبول: إنشاء نتيجة مع درجة ثقة خلال زمن مقبول.
- Story: كمعلم أريد مراجعة الحالات منخفضة الثقة أولًا.
  - معيار قبول: قائمة مراجعة مرتبة حسب درجة الثقة.

## Epic 3: المكتبة الحية
- Story: كطالب أريد إجابة مرتبطة بمصدر رسمي من المنهج.
  - معيار قبول: كل إجابة تتضمن مرجعًا واضحًا.

## Epic 4: ملف الطالب الموحد
- Story: كمنصة أريد تجميع أداء الطالب من كل الخدمات.
  - معيار قبول: تحديث الملف فور وصول الأحداث الموحدة.

## Epic 5: لوحة ولي الأمر
- Story: كولي أمر أريد رؤية التقدم والمخاطر بسرعة.
  - معيار قبول: عرض مؤشرات أسبوعية مبسطة وقابلة للفهم.
