# KPIs تشغيلية وتعليمية

## تشغيلية
- توفر الخدمة (Availability)
- متوسط زمن الاستجابة (P95 Latency)
- معدل الأخطاء (Error Rate)
- زمن معالجة رفع الورقة حتى النتيجة

## تعليمية
- معدل التبني حسب الدور
- دقة OCR على الخط اليدوي العراقي
- نسبة الحالات منخفضة الثقة
- تحسن الأداء عبر الزمن
- معدل الاحتفاظ بالمستخدمين
