# 07) تجربة المستخدم

## المعلم
- واجهة سريعة للتصحيح الجماعي.
- إبراز الحالات منخفضة الثقة أولًا.

## الطالب
- تجربة موجهة للعلاج التعليمي لا لعرض الدرجة فقط.
- توصيات مباشرة مرتبطة بنقاط الضعف.

## ولي الأمر
- لوحة مبسطة تبرز التقدم والمخاطر بنظرة واحدة.

## اللغة والمحتوى
- دعم العربية بالكامل.
- اتساق المصطلحات مع المنهج العراقي.
