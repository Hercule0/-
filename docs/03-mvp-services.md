# 03) تصميم الخدمات الأساسية (MVP)

## 1) Identity & Accounts
- تسجيل الدخول الموحد.
- إدارة الأدوار والصلاحيات.
- إدارة الجلسات.

## 2) Yara Service
- رفع ورقة الطالب.
- OCR وتحليل الإجابات.
- تقييم أولي + درجة ثقة.
- إرسال الحالات منخفضة الثقة لمراجعة المعلم.

## 3) Live Library Service
- فهرسة المنهج العراقي.
- استرجاع معرفي مقيّد بالمحتوى الرسمي.
- إجابات مرتبطة بمراجع واضحة.

## 4) Student Profile Service
- تخزين الأداء ونقاط الضعف.
- توحيد سجل التفاعل من جميع الخدمات.
- تغذية لوحة ولي الأمر بمؤشرات دقيقة.

## 5) Parent Dashboard Service
- عرض مؤشرات تقدم موحدة ومبسطة.
- تنبيهات للمخاطر التعليمية.
- عرض اتجاه الأداء عبر الزمن.
