# 09) الاختبارات وضمان الجودة

## أنواع الاختبارات
- اختبارات وحدات لكل خدمة.
- اختبارات تكامل بين API Gateway والخدمات.
- اختبارات قبول لسيناريوهات المستخدم الأساسية.
- اختبارات أداء لرفع الصور والاستعلامات المعرفية.

## سيناريوهات قبول أساسية
- رفع ورقة عبر يارا ثم اعتماد/تعديل النتيجة.
- استعلام طالب من المكتبة مع إرجاع مصدر رسمي.
- ظهور أثر التقدم في لوحة ولي الأمر.
