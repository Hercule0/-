# 06) الأمن والامتثال

## ضوابط أساسية
- RBAC دقيق للأدوار التعليمية.
- تشفير البيانات أثناء النقل (TLS) والتخزين (at-rest).
- سجلات تدقيق لكل عمليات التصحيح وتعديل الدرجات.

## الخصوصية
- سياسات واضحة لبيانات الطلبة وصور الدفاتر.
- تقليل البيانات المخزنة إلى الحد الأدنى المطلوب.
- صلاحيات وصول مبنية على الحاجة الفعلية.
